---

Load the following skills before doing any frontend work in this session:

- Invoke the `ant:nextjs-ppr` skill — Partial Prerendering patterns, static/dynamic boundaries, Suspense granularity
- Invoke the `ant:next-best-practices` skill — Next.js file conventions, RSC boundaries, async APIs, metadata, error handling
- Invoke the `ant:react-best-practices` skill — React and Next.js performance optimization guidelines
- Invoke the `ant:composition-patterns` skill — React composition patterns, compound components, avoiding boolean prop proliferation

After loading all four skills, confirm you're ready and ask what to work on.
