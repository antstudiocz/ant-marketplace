# Phase Close Template

```text
Phase / owner / status:
Inputs and approved scope:
Work completed and changed artifacts:
Decisions and rationale refs:
Evidence records:
Review status and open findings:
Open questions / blockers / residual risks:
Active children and write scopes:
Next phase and first safe action:
Files to read first:
Must not assume:
State/event updates made or exact updates for parent:
```

A phase does not close until lifecycle transition conditions and required structured-state updates are satisfied.
